# Algebra Basics

## What is Algebra?
Algebra is a branch of mathematics that uses symbols (usually letters) to represent unknown quantities. It allows us to write general rules and solve problems where some values are unknown.

## Variables and Expressions

### Variables
A variable is a letter that represents an unknown value. Common choices: x, y, z, n.

### Expressions
An algebraic expression is a combination of variables, numbers, and operations.
- Examples: 3x + 5, 2y² - 4y + 1, (a + b)/2

### Evaluating Expressions
To evaluate: substitute the value of the variable and calculate.
- If x = 3, then 2x + 1 = 2(3) + 1 = 7

## Linear Equations

### What is a Linear Equation?
A linear equation has the form **ax + b = c**, where a, b, c are constants and x is the unknown. The highest power of the variable is 1 (hence "linear").

### Solving Linear Equations
The goal is to isolate the variable by performing the same operation on both sides.

**Example:** Solve 3x + 7 = 16
1. Subtract 7 from both sides: 3x = 9
2. Divide both sides by 3: x = 3

**Example:** Solve 2(x - 4) = 10
1. Expand brackets: 2x - 8 = 10
2. Add 8 to both sides: 2x = 18
3. Divide by 2: x = 9

### Equations with Variables on Both Sides
**Example:** Solve 5x - 3 = 2x + 9
1. Subtract 2x from both sides: 3x - 3 = 9
2. Add 3 to both sides: 3x = 12
3. Divide by 3: x = 4

## Quadratic Equations

### What is a Quadratic Equation?
A quadratic equation has the form **ax² + bx + c = 0**, where a ≠ 0. The highest power of the variable is 2.

### Method 1 — Factoring
Find two numbers that multiply to ac and add to b.

**Example:** Solve x² + 5x + 6 = 0
- Need two numbers that multiply to 6 and add to 5: 2 and 3
- Factor: (x + 2)(x + 3) = 0
- Either x + 2 = 0 → x = -2, or x + 3 = 0 → x = -3
- Solutions: x = -2 or x = -3

### Method 2 — The Quadratic Formula
For any quadratic ax² + bx + c = 0:

**x = (-b ± √(b² - 4ac)) / 2a**

**Example:** Solve 2x² - 4x - 6 = 0
- a = 2, b = -4, c = -6
- Discriminant: b² - 4ac = 16 - 4(2)(-6) = 16 + 48 = 64
- x = (4 ± √64) / 4 = (4 ± 8) / 4
- x = 12/4 = 3, or x = -4/4 = -1
- Solutions: x = 3 or x = -1

### The Discriminant
The value b² - 4ac tells us the nature of the solutions:
- **b² - 4ac > 0** → two distinct real solutions
- **b² - 4ac = 0** → one repeated real solution
- **b² - 4ac < 0** → no real solutions (complex roots)

### Method 3 — Completing the Square
Rewrite the quadratic in the form (x + p)² = q.

**Example:** Solve x² + 6x + 5 = 0
1. Move constant: x² + 6x = -5
2. Add (6/2)² = 9 to both sides: x² + 6x + 9 = 4
3. Factor left side: (x + 3)² = 4
4. Take square roots: x + 3 = ±2
5. Solutions: x = -1 or x = -5

## Systems of Equations

### What is a System?
A system of equations is two or more equations with the same variables. The solution is the set of values that satisfies all equations simultaneously.

### Method 1 — Substitution
**Example:** Solve the system: y = 2x + 1 and 3x + y = 11
1. Substitute y = 2x + 1 into second equation: 3x + (2x + 1) = 11
2. Simplify: 5x + 1 = 11 → 5x = 10 → x = 2
3. Substitute back: y = 2(2) + 1 = 5
4. Solution: x = 2, y = 5

### Method 2 — Elimination
**Example:** Solve: 2x + 3y = 12 and 4x - 3y = 6
1. Add the equations: 6x = 18 → x = 3
2. Substitute: 2(3) + 3y = 12 → 6 + 3y = 12 → y = 2
3. Solution: x = 3, y = 2

## Inequalities

### Linear Inequalities
Solved like equations, but with one rule: **when multiplying or dividing by a negative number, flip the inequality sign**.

**Example:** Solve -2x + 3 > 7
1. Subtract 3: -2x > 4
2. Divide by -2 (flip sign): x < -2

### Graphing Inequalities
- **>** or **<**: dashed boundary line (not included)
- **≥** or **≤**: solid boundary line (included)
- Shade the region that satisfies the inequality

## Functions

### Definition
A function is a rule that assigns exactly one output to each input. Written f(x) = ...

### Domain and Range
- **Domain**: all valid input values (x)
- **Range**: all possible output values f(x)

### Common Functions
- Linear: f(x) = mx + b (straight line, slope m, y-intercept b)
- Quadratic: f(x) = ax² + bx + c (parabola)
- Absolute value: f(x) = |x| (V-shape)

### Function Notation
f(x) = 3x - 2
- f(4) = 3(4) - 2 = 10
- f(-1) = 3(-1) - 2 = -5
